# Phase 3: AI-Powered Chatbot

This directory contains the implementation of the AI-powered chatbot component for the Todo application.

## Contents
- `chatbot/` - The main chatbot implementation
- `README.md` - Detailed documentation for this phase

## Purpose
Phase 3 focuses on implementing an intelligent chatbot that allows users to manage their todos using natural language commands. This component integrates with the backend API and provides a conversational interface to the todo management system.
